1. Introduction

This document explains the statistical information contained in the Archive Report Cards, generated by archiveReport.php.

2. Archive Diversity

For the subject and type fields, these percentages show:

    Diversity = (Distinct code values / Number instances of tag) * 100

This gives an indication of the diversity of the information held by the archive.

3. Metadata Quality

Graph showing the frequency of record scores within the archive. For an explanation of how record scores are calculated, see documentation.html.

4. Core Tags Per Record

The percentage of records which have n of the core tags present at least once.

5. Core Tag Usage

Percentage of records which contain the named tag at least once. Red highlights tags which are not used in all records from this archive.

6. Code Usage

Displays the number of times a tag (which has an associated code attribute) was used by the archive, and the percentage of those tags which used a code attribute. Red highlights tags which did not contain code attributes in all instances of that tag.

7. Tag and Code Usage

Number of times a tag is used. Where applicable, the number of times that a code attribute is used with that tag. Red highlights tags which do not use attributes in all instances of that tag.

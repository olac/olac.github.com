0. TERMINOLOGY

OAI-PMH: The Open Archives Initiative (OAI) Protocol for Metadata Harvesting, [http://www.openarchives.org/OAI_protocol/openarchivesprotocol.html]. OAI data provider: Any metadata repository which participates in the OAI-PMH by providing XML documents in response to protocol requests.

OAI identifier: Identifiers of the form oai:... which are used in the Open Archives Initiative to identify OAI data providers and metadata records.

OLAC archive: Any OAI data provider which is registered at www.language-archives.org.

OLAC metadata: Any XML document generated by an OLAC archive as part of the OAI-PMH, and any modified versions including translations, aggregations and partial documents, and any conversions into HTML or any other format.

PREPARATION

Here are various ways in which participants can prepare for the workshop:

  1. Review the proposed standards:
    • OLAC Metadata
    • OLAC Process
    • OLAC Protocol
  2. Review the recommendations:
    • Recommended metadata extensions
    • OLAC Access Vocabulary
    • OLAC Linguistic Field Vocabulary
    • OLAC Role Vocabulary
    • OLAC Linguistic Type Vocabulary
    • OLAC Discourse Type Vocabulary
  3. Review the notes:
    • A Gentle Introduction to Metadata
    • Third party extensions
    • A query facility for selective harvesting of OLAC metadata
    • Viser: A virtual service provider for displaying selected OLAC metadata
    • Specifications for an OLAC metadata display format and an OLAC-to-OAI_DC crosswalk
  4. Miscellaneous:
    • Read up on OLAC technical infrastructure (12-page overview).
    • Read the OLAC 1.0 XML Schemas and examples
    • Review the OLAC-Implementers mailing list archive
    • Review the METADATA mailing list archive
